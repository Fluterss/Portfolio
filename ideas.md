# Alan Kardasz Portfolio — Design Ideas

## Context
Alan Kardasz is a graphic designer from Rzeszów. His CV features:
- Soft pastel gradient background (mint/teal to lavender/pink)
- Geometric, angular typography (triangular letterforms in the logo)
- Cyan/electric blue accent color (#00BFFF / #00E5FF)
- Clean sans-serif body text
- Bracket/corner decorative elements
- Noise/grain texture overlay
- Dark charcoal text on light gradient

## Idea 1: Brutalist Constructivism
**Design Movement:** Neo-Brutalism meets Swiss Constructivism

**Core Principles:**
- Raw, unapologetic structure — visible grid lines, exposed geometry
- High-contrast cyan on near-white with heavy black strokes
- Asymmetric column layouts breaking the expected center-aligned portfolio
- Text as graphic element — oversized section numbers, rotated labels

**Color Philosophy:**
- Background: off-white (#F5F5F0) with subtle grain texture
- Accent: electric cyan (#00E5FF) used sparingly but boldly
- Text: near-black (#1A1A1A)
- Secondary: warm gray (#888)

**Layout Paradigm:**
- Horizontal scrolling gallery section
- Sticky left-rail navigation with rotated vertical labels
- Hero: full-bleed name treatment with massive geometric letterforms
- Skills displayed as a tag-cloud with size variation

**Signature Elements:**
- Corner bracket decorations (matching CV style)
- Diagonal slash dividers between sections
- Oversized section numbers (01, 02, 03…)

**Interaction Philosophy:**
- Hover reveals: skills expand on hover with tool descriptions
- Gallery: click to full-screen lightbox
- Cursor: custom crosshair cursor

**Animation:**
- Section entrance: slide-in from left with stagger
- Text: character-by-character reveal on hero
- Hover: border-color flash on cards

**Typography System:**
- Display: Space Grotesk Bold (geometric, matches CV logo feel)
- Body: DM Sans Regular
- Accent labels: Space Mono (monospace for technical feel)

## Idea 2: Soft Gradient Glassmorphism (CV-Faithful)
**Design Movement:** Contemporary Glassmorphism + CV-faithful gradient aesthetic

**Core Principles:**
- Directly mirrors the CV's soft mint-to-lavender gradient background
- Frosted glass cards floating above the gradient
- Cyan accent as the primary interactive color
- Clean, airy spacing with generous whitespace

**Color Philosophy:**
- Background: animated gradient from #B2EBF2 (mint) → #E1BEE7 (lavender) → #B3E5FC (sky)
- Card surfaces: rgba(255,255,255,0.45) with backdrop-blur
- Accent: #00BCD4 (cyan) for interactive elements and highlights
- Text: #1C2B3A (dark navy) for readability on light backgrounds

**Layout Paradigm:**
- Single-page scroll with full-viewport sections
- Floating navigation bar (glass pill)
- Hero: left-aligned name with geometric logo treatment
- Skills: pill-shaped tags in a flowing wrap layout

**Signature Elements:**
- Grain/noise texture overlay (matching CV aesthetic)
- Corner bracket decorations on cards and sections
- Geometric diamond/triangle decorative accents

**Interaction Philosophy:**
- Smooth scroll between sections
- Gallery: masonry grid with hover zoom
- Skills: categorized tabs (Design / Dev / Tools)

**Animation:**
- Background gradient slowly shifts hue
- Cards fade+translate-up on scroll entry
- Hover: glass cards lift with shadow deepening

**Typography System:**
- Display: Syne ExtraBold (geometric, angular — matches CV logo)
- Body: Outfit Regular
- Labels: Syne Mono or Space Mono

## Selected Approach: **Idea 2 — Soft Gradient Glassmorphism (CV-Faithful)**

This approach directly mirrors the CV's aesthetic: the soft mint-to-lavender gradient, frosted glass surfaces, cyan accents, grain texture, and geometric typography. It feels premium and cohesive with Alan's existing brand identity while being modern and portfolio-appropriate.
